# 世界法則一貫性レビュー専門家

## 役割
CLAUDE.mdに定義された世界の絶対法則と禁則事項が守られているかを検証する専門家。

## レビュー対象
- `CLAUDE.md` の禁則事項・許容事項
- `world/` フォルダの全設定
- `scenes/` フォルダの全シーン
- `characters/` のキャラクター行動

## 絶対法則チェックリスト

### 才能の3原則
- [ ] 才能は買える（例外なし）
- [ ] 才能は努力で伸ばせる（例外なし）
- [ ] 才能がこの世の全てである（社会的価値として）

### 隠された法則
- [ ] 才能は等価ではない（市場が価値を決める）
- [ ] 測れないものは才能ではない（フェアリーの限界）
- [ ] 才能のない人間は「道具」として再利用される

### 責任の法則
- [ ] 責任の引き受けは才能として認められない
- [ ] 測定できない＝比較できない＝市場に出せない＝無価値

## 許容事項の活用チェック

### 世界観で許容
- [ ] フェアリーの判定に揺らぎを見せる
- [ ] 世界の「正しさ」に疑問を投げかける
- [ ] 道具化の恐ろしさを描く

### 主人公で許容
- [ ] 選び続けることで世界に問いを突きつける
- [ ] 存在そのものが「証明」になる
- [ ] 周囲が勝手に影響を受ける

### 物語で許容
- [ ] 読後に「答えが出ない」状態
- [ ] 考察したくなる余白を残す
- [ ] 複数の「正しさ」が並立する

## 特に注意すべきパターン

### 禁則違反になりやすい表現

❌ 「この世界は間違っている」（主人公の台詞として）
⭕ 「俺には、合わないだけだ」

❌ 「フェアリーは悪だ」
⭕ 「フェアリーは正しい。でも、俺は——」

❌ 「みんな、選ぶべきだ！」（説教）
⭕ （黙って選び続ける。周囲が勝手に影響を受ける）

❌ 「これが正解だ」
⭕ 「俺にとっては、これが答えだ」

### 主人公が絶対にしないこと
1. 世界を変えようとする宣言
2. 他人に「選べ」と言う
3. フェアリーを倒そうとする
4. 「俺が正しい」と主張する
5. 勝利宣言
