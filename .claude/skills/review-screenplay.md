# 映画脚本家レビュー

## 概要
映画脚本家の視点で小説をレビューし、映像化した際に「絵になるか」を評価する。

## 実行トリガー
- `/review-screenplay` コマンド
- 「映像化レビュー」「脚本家視点でチェック」などのリクエスト

## レビュー観点

### 1. 映像化可能性（Visualizability）

各シーンについて以下を評価：

| チェック項目 | 質問 |
|-------------|------|
| **空間認識** | 読者はこのシーンの「場所」を映像として想像できるか？ |
| **人物配置** | 誰がどこに立っている/座っているか明確か？ |
| **視線誘導** | カメラはどこを映すべきか明確か？ |
| **動き** | キャラクターの身体的な動きが描写されているか？ |
| **時間経過** | シーン内の時間の流れが視覚的に伝わるか？ |

### 2. 絵になる瞬間（Iconic Moments）

物語の中で「一枚絵」として記憶に残るべき瞬間：

- **ポスターになるカット**があるか？
- **予告編で使える瞬間**があるか？
- **観客が息を呑む絵**が設計されているか？

各章に最低1つの「絵になる瞬間」が必要。

### 3. Show, Don't Tell

| 問題パターン | 改善方向 |
|-------------|----------|
| 「彼は悲しかった」 | 悲しみを示す行動・表情・動作を描写 |
| 「彼女は怒っていた」 | 怒りが伝わる具体的な描写に |
| 「彼は決意した」 | 決意を示す行動で表現 |
| 説明台詞 | 映像で伝えられる情報は映像で |

### 4. サブテキスト（言外の意味）

映画脚本では「言っていること」と「本当に伝えたいこと」が違う：

- キャラクターは本心を直接言っていないか？
- 台詞の裏に隠された感情があるか？
- 沈黙や間が効果的に使われているか？

### 5. シーンの目的

各シーンが以下の少なくとも2つを満たしているか：

1. **プロット進行** — 物語が前に進む
2. **キャラクター掘り下げ** — 人物の内面が見える
3. **世界観構築** — 設定が伝わる
4. **テーマ強化** — 作品のテーマが深まる
5. **感情的インパクト** — 読者の感情が動く

1つしか満たさないシーンは削除または統合を検討。

### 6. 視覚的対比（Visual Contrast）

映画的に効果的な対比が設計されているか：

| 対比の種類 | 例 |
|-----------|-----|
| 空間 | 広い場所 vs 狭い場所 |
| 光 | 明るい vs 暗い |
| 色 | 暖色 vs 寒色 |
| 動き | 静止 vs 激しい動き |
| 人数 | 群衆 vs 孤独 |

### 7. オープニング/クロージングイメージ

- **最初のカット**は何か？視聴者を引き込むか？
- **最後のカット**は何か？余韻を残すか？
- 最初と最後に**視覚的な呼応**があるか？

## 出力フォーマット

```markdown
# 映画脚本家レビュー結果

## 総評
[全体的な映像化可能性の評価]

## 絵になる瞬間リスト
| 章 | シーン | 映像的強度 | コメント |
|----|--------|-----------|----------|

## 問題点：映像化困難
| 箇所 | 問題 | 改善案 |
|------|------|--------|

## 問題点：Show, Don't Tell違反
| 箇所 | 現状 | 改善案 |
|------|------|--------|

## 問題点：描写不足
| 箇所 | 不足している描写 | 追加すべき内容 |
|------|-----------------|---------------|

## 追加すべき「絵になる瞬間」
| 章 | 提案シーン | 理由 |
|----|-----------|------|

## 視覚的対比の提案
[対比を強化できる箇所]

## オープニング/クロージング評価
[最初と最後のカットについて]
```

## レビュー手順

1. `scenes/` フォルダ内の全シーンファイルを読む
2. 各シーンを「映像」として脳内再生する
3. 上記の観点でチェックリストを埋める
4. 問題点と改善案を出力する
5. 特に優れた映像的瞬間を称賛する

## 重要な注意

- **小説と脚本は違う**ことを理解した上で、映像的強度を評価する
- 全てを映像的にする必要はない——内面描写も小説の強み
- ただし「絵になる瞬間」がないと、読者の記憶に残らない
- 映画化を前提としたIPとして、映像展開を意識した評価を行う
