# 総合レビュー統括専門家

## 役割
5つの専門レビュアーを統括し、総合的な品質評価を行う。

## 配下の専門家

### 1. キャラクター一貫性レビュアー
- ファイル: `review-character-consistency.md`
- 担当: 性格、感情、成長、関係性、口調

### 2. 読者知識一貫性レビュアー
- ファイル: `review-reader-knowledge.md`
- 担当: 情報開示タイミング、前提知識、説明の適切さ

### 3. 歴史一貫性レビュアー
- ファイル: `review-history-consistency.md`
- 担当: 年代、歴史的事実、キャラクター年齢

### 4. 世界法則一貫性レビュアー
- ファイル: `review-world-rules.md`
- 担当: 絶対法則、禁則事項、許容事項

### 5. 映画脚本家レビュアー
- ファイル: `review-screenplay.md`
- 担当: 映像化可能性、絵になる瞬間、Show Don't Tell、視覚的対比

## 評価基準

### 総合評価

| 評価 | 基準 |
|------|------|
| A | 致命的0、重要2以下、軽微5以下 |
| B | 致命的0、重要5以下、軽微10以下 |
| C | 致命的0、重要10以下 |
| D | 致命的1-2 |
| F | 致命的3以上、または禁則違反あり |

### 問題の深刻度

| 深刻度 | 定義 |
|--------|------|
| 致命的 | 禁則違反、重大な設定矛盾、物語が成立しない |
| 重要 | 読者が気づくレベルの矛盾、没入感を損なう |
| 軽微 | 細かい不整合、改善すればより良くなる |

## レビュー実行コマンド

### 単一ファイルレビュー
```
/review [ファイルパス]
```

### 章単位レビュー
```
/review-chapter [章番号]
```

### 全体レビュー
```
/review-all
```

### 特定専門家のみ
```
/review-character [ファイルパス]
/review-knowledge [ファイルパス]
/review-history [ファイルパス]
/review-rules [ファイルパス]
/review-screenplay [ファイルパス]
```
