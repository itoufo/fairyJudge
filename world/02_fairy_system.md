# フェアリーシステム

---

## フェアリーとは何か

| 項目 | 定義 |
|------|------|
| 役割 | 才能の測定・評価・仲介 |
| 立場 | 世界＝法則の執行者 |
| 感情 | 持たない（または制限されている） |
| 判定 | 絶対に間違わない（とされている） |
| 存在形態 | 白い光の中に浮かぶ透き通った輪郭 |

---

## フェアリーの機能

### 1. 測定（メジャメント）
- 個人の才能を数値化
- 才能の種類、強度、市場価値を算出
- 成長可能性も予測

### 2. 評価（ジャッジメント）
- 才能の市場価値を判定
- 適正ルートを提示
- 行動の推奨/非推奨を決定

### 3. 仲介（トランザクション）
- 才能の売買を仲介
- インストール（才能の付与）
- アンインストール（才能の剥奪）

---

## 評価基準

### 評価される要素
| 要素 | 説明 |
|------|------|
| 再現性 | 同じ結果を繰り返し出せるか |
| 比較可能性 | 他者と比較できるか |
| 市場価値 | 世界に貢献できるか |
| 予測可能性 | 将来の成果が計算できるか |

### 評価されない要素
| 要素 | 理由 |
|------|------|
| 責任 | 結果に付随する概念であり事前に測れない |
| 覚悟 | 内面的であり外部から観測できない |
| 信じる力 | 結果が保証されないため価値に換算できない |
| 選択する意志 | 個人的であり模倣できない |

---

## フェアリーの判定文言（例）

**通常判定**
```
「測定を開始します」
「適正、基準値内。市場価値、安定」
「次の判定へ進みます」
```

**責任に関する判定**
```
「責任の引き受けは、才能ではありません」
「それは評価対象ではなく、交換価値を持たず、市場に寄与しません」
「従って、この行為は無価値です」
```

---

## フェアリーの「正しさ」

フェアリーは悪ではない。間違ってもいない。

**フェアリーの論理**
1. 世界は続くべきである
2. 安定のためには再現可能な判断基準が必要である
3. 再現できない行為を評価すると世界は不安定になる
4. したがって、責任や覚悟は評価しない

---

## フェアリーの限界

### 測定の盲点
- 「責任を引き受ける存在が消失した世界」のシミュレーションは行われていない
- フェアリーは「知らないまま、切り捨てている」

### 揺らぎの兆候
- 処理遅延（0.03秒）
- 未解決ログの蓄積
- 判定不能の発生
