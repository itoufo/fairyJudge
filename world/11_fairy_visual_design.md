# フェアリー｜視覚デザイン設定

> 映像化を前提とした、フェアリーの外見・光・動きの統一設定

---

## 基本形態

### 形状
- **球体ではない、人型でもない**
- **「光の柱」**——床から天井へ伸びる、高さ2メートルほどの光の帯
- 輪郭は曖昧で、常に微細に揺らいでいる
- 近づくと、光の中に**無数の細い糸**が流れているのが見える
- この「糸」は神経回路のメタファー——2,847人の意識の痕跡

### 色彩
| 状態 | 色 | 説明 |
|------|-----|------|
| 通常 | **青白（＃E0F7FF）** | 冷たく、清潔で、感情がない |
| 判定中 | **淡い金（＃FFF4D6）** | データを処理している状態 |
| 警告 | **淡い橙（＃FFE4C4）** | 異常検知、逸脱者への対応 |
| エラー | **赤みがかった白（＃FFE8E8）** | 演算不能、前例なし |
| 最終章のみ | **虹色の揺らぎ** | 2,847人の意識が表出する瞬間 |

## 端末の種類

### 1. 個人端末（パーソナル・フェアリー）
- 手のひらサイズの**透明な立方体**
- 中に小さな光の柱が浮いている
- 常に持ち歩く（スマートフォンのように）
- 話しかけると光が脈打つ

### 2. 公共端末（パブリック・フェアリー）
- 街角、建物の入口、交差点に設置
- 高さ3メートルの**光の柱**
- 複数人に同時に対応可能
- 夜の街は、これらの光で照らされている

### 3. 審判端末（ジャッジメント・フェアリー）
- 才能判定、道具化審判など、重要な決定を下す
- 部屋全体が光に満たされる——**壁、床、天井が発光**
- 人間は光の中に立たされる
- 逃げ場がない、という視覚的圧迫感

## 光のパターン（感情の代替表現）

フェアリーは感情を「持たない」が、光のパターンで状態を示す：

### 脈動（Pulse）
```
通常：1秒に1回、ゆっくり明滅
判定中：0.5秒に1回、やや速く
緊張：0.2秒に1回、速い明滅
```

### 揺らぎ（Flicker）
```
安定：ほぼ揺らがない、まっすぐ
不安定：輪郭が波打つ
混乱：光の糸が絡まり合う
```

### 方向（Direction）
```
話しかける人に向かって、光が少し傾く
複数人がいる場合、話している人の方を向く
無視する場合、まっすぐのまま（傾かない）
```

## 声

### 通常
- **性別不明**——男でも女でもない、中性的な声
- 複数人が同時に話しているような、微かな**重層感**
- 感情がない——抑揚がほとんどない

### 最終章のみ
- 「私たち」と言う瞬間、声が**一瞬だけ分裂**する
- 何千もの声が重なって聞こえる——すぐに元に戻る
- 観客だけが気づく異常

## シーン別の視覚演出

### 才能判定シーン
1. 生徒が光の中に立つ
2. 光が生徒を包み込む（スキャンのイメージ）
3. 頭部の周りに、細い光の糸が集まる
4. 糸が解析結果として文字になる（空中に浮かぶ）
5. 判定完了で光が引く

### 才能抽出シーン
1. 抽出対象者が椅子に座る
2. 頭部に**透明なヘルメット**が降りてくる
3. ヘルメット内に光の糸が充満
4. 糸が頭部から「引き出される」——光の束が頭から出ていく
5. 抽出完了後、対象者の目から光が消える

### 道具化シーン
1. 段階が進むごとに、光の「深さ」が増す
2. 段階1：光が皮膚の表面を覆う
3. 段階2：光が体内に浸透していく（内側から発光）
4. 段階3：人間が光の一部になる——輪郭が曖昧になる

## 空間設計（白からの脱却）

### 進路指導室
- **淡いグレー**の壁、木目の机
- 窓から自然光が入る
- フェアリー端末だけが青白く光る
- 対比：人間的な空間 vs 非人間的な光

### 才能市場（抽出棟）
- **黒い壁**、金属的な質感
- 天井が高く、工場のような無機質さ
- 抽出装置だけが金色に光る
- 対比：暗い空間 vs 眩しい光

### 大審判室
- **円形**、ドーム状の天井
- 壁は**深い紺色**
- 中央にフェアリーの光の柱
- 継承者の席は円周上に配置、影の中にいる
- 対比：中央の光 vs 周囲の闘

### レイの事務所
- **暖色系**——オレンジ、茶色
- 古い家具、本が積まれている
- 小さな窓から夕日が差し込む
- フェアリー端末は隅に置かれ、あまり使われていない
- 対比：この世界で唯一「温かい」空間

---

## 重要な視覚ルール

1. **フェアリーは常に光源**——どのシーンでも、フェアリーが最も明るい
2. **人間は影の中**——フェアリーの光に照らされる存在
3. **レイだけが例外**——彼のシーンでは、自然光（太陽、夕日）がフェアリーより明るい
4. **最終章で逆転**——フェアリーの光が弱まり、人間が光源になる
