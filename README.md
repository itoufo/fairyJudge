# フェアリージャッジ (Fairy Judge)

> 「才能があっても、正解があっても、それでも人は選び続けていい」

---

## 概要

**フェアリージャッジ**は、才能が全てを決める世界で、
「選ぶこと」を奪われた主人公が、努力で這い上がる物語。

世界を壊す革命ではない。
ただ、「選び続ける人間がいる」という事実を証明する話。

---

## 世界の3原則

1. **才能は買える**
2. **才能は努力で伸ばせる**
3. **才能がこの世の全てである**

---

## テーマ

> 「世界の正しさをめぐる物語」

**責任を引き受ける力**は、才能として認められない。
**選び続ける意志**は、市場価値を持たない。

でも、それがなければ、人は「生きた」と言えるのか？
