## 外見

- 金髪、整った顔立ち
- 長身、均整のとれた体格
- 完璧な笑顔（だが目が笑っていない）
- 高級な服装、隙がない
- オーラがある——だが、どこか空虚

## 性格の核心

### 一言定義
> **「全ての才能を持っているが、何も選んだことがない男」**

### 内面の真実
- 生まれた時から「完璧」だった
- 何をしても成功する。失敗したことがない
- フェアリーの推奨は常に「何でもいい」——全てが最適解だから
- **一度も「選ぶ」必要がなかった**
- 成功しているのに、空虚を感じている
- **レイと出会い、初めて「羨ましい」と感じる**

## 背景

### 「完璧」の呪い
- 才能が全て最上位＝何を選んでも成功
- 「選ぶ」プレッシャーがない
- 失敗したことがない＝失敗の恐怖も知らない
- **「選んだ」という感覚がない**

### 英雄として
- 社会の象徴、フェアリーシステムの「成功例」
- 誰もが羨む存在
- しかし本人は——

> 「俺は何も選んでいない。全部、勝手にうまくいっただけだ」

## 感情曲線

### 第8章：出会い
```
感情：好奇心 → 困惑 → 衝撃
出来事：レイの存在を知り、接触する
内面：「才能もないのに、なぜあいつは——」
キーフレーズ：「お前は、俺にないものを持っている」
```

### 第9章：理解
```
感情：羨望 → 自己嫌悪 → 問い
出来事：レイの道具化を見届ける
内面：「俺は成功している。でも、あいつの方が——生きている」
キーフレーズ：「俺は生まれてから一度も、選んだことがない」
```

### 最終章：変化の兆し
```
感情：迷い → 微かな決意
出来事：フェアリーの「質問」を聞く
内面：「俺も、何かを選んでみたい」
→ エピローグで「英雄」の地位を捨てる伏線
```

| ソウマ | レイ |
|--------|------|

ソウマは黙った。

ソウマ：「……俺は、その感覚がわからない」

　　　　 だから——選ぶ必要がない」

レイ：「……」

# キャラクター配置の意図

## 対比軸

1. **才能の軸**
   - ソウマ（全才能） ↔ レイ（才能なし） ↔ リン（測定不能）

2. **選択の軸**
   - マコト（従う） ↔ レイ（選ぶ） ↔ ナギ（道具化後も選ぶ）

3. **疲弊の軸**
   - カイ（疲弊する） ↔ レイ（平気） ↔ ソウマ（選ばない）
