## 口癖・特徴的なフレーズ

### レイ

| 口癖 | 使用場面 |
|------|----------|
| 「俺が選んだからだ」 | 理由を聞かれた時 |
| 「だから何だ」 | 脅しや警告に対して |
| 「……」（沈黙） | 答えを持っていない時、考えている時 |
| 「お前が決めろ」 | 他人の選択を尊重する時 |
| 「失敗しても、俺の失敗だ」 | 覚悟を示す時 |

**話し方の特徴**：
- 短い文。余計なことを言わない
- 質問には質問で返すことがある
- 説教しない。自分の意見を押し付けない
- 沈黙が多い。でも、逃げの沈黙ではない

### ユキ

| 口癖 | 使用場面 |
|------|----------|
| 「私には関係ない」 | 自分を守る時（本心ではない） |
| 「……わかんない」 | 感情が溢れそうな時 |
| 「母さんの才能だから」 | 自己否定 |
| 「私、初めて自分で選んだ」 | 第8章の決断 |

**話し方の特徴**：
- 感情を抑えた話し方
- でも、レイの前では本音が漏れる
- 「……」が多い（言葉を探している）
- 決断した後は、はっきり話す

### カイ

| 口癖 | 使用場面 |
|------|----------|
| 「俺には関係ない才能だ」 | 戦闘能力について |
| 「疲れた……」 | 選択疲弊を感じている時 |
| 「でも、やめない」 | 決意を示す時 |
| 「お前は、なんで平気なんだ」 | レイへの問いかけ |

**話し方の特徴**：
- 穏やか。声を荒げない
- 外見とのギャップ（大柄だけど優しい口調）
- 弱音を吐くことがある（隠さない）

### リン

| 口癖 | 使用場面 |
|------|----------|
| 「私は何もしてません」 | 感謝された時（謙遜） |
| 「ただ、聞いてるだけです」 | 自分の才能について |
| 「私は、ここにいます」 | 誰かを支える時 |
| 「道具になりたくない」 | 恐怖を語る時 |

**話し方の特徴**：
- 丁寧語が基本
- でも、感情的になると砕けた口調になる
- 「……」は不安を示す
- 笑顔で話すことが多い

### 老師

| 口癖 | 使用場面 |
|------|----------|
| 「考えろ」 | 答えを教えない時 |
| 「俺は臆病だった」 | 過去を語る時 |
| 「お前なら——」 | レイに期待を込めて |
| 「時が来れば、わかる」 | 真実を隠している時 |

**話し方の特徴**：
- 短い。無駄がない
- 質問に直接答えない
- 「……」は考えている、または迷っている
- 死の直前は、言葉が増える

### ソウマ

| 口癖 | 使用場面 |
|------|----------|
| 「どれでも同じだ」 | 選択を求められた時 |
| 「成功する。それだけだ」 | 自分について |
| 「……空っぽだ」 | 本音が漏れる時（稀） |
| 「君は、なぜそんなに必死なんだ」 | レイへの問い |

**話し方の特徴**：
- 完璧な敬語も使えるが、レイには砕けた口調
- 余裕がある。焦らない
- でも、空虚さが滲む瞬間がある

### マコト

| 口癖 | 使用場面 |
|------|----------|
| 「フェアリーの推奨通りです」 | 正当化する時 |
| 「なぜ従わないの」 | 理解できない時 |
| 「それが正しいから」 | 信念を語る時 |
| 「私は間違っていない」 | 追い詰められた時 |

**話し方の特徴**：
- 丁寧だが、冷たい
- 「正しい」「効率的」「推奨」を多用
- 感情的になると声が高くなる

### セラフィム

| 口癖 | 使用場面 |
|------|----------|
| 「システムを守ることが、人類を守ることだ」 | 信念 |
| 「お前の『俺』に、何の価値がある」 | レイへの問い |
| 「エルドアを忘れたのか」 | 恐怖を煽る時 |
| 「逸脱者は——」 | 判定を下す時 |

**話し方の特徴**：
- 威厳がある。冷たい
- 感情を表に出さない（努力して）
- 追い詰められると声が震える

### エリオス

| 口癖 | 使用場面 |
|------|----------|
| 「興味深い」 | 何かを観察する時 |
| 「君なら——」 | 可能性を示唆する時 |
| 「システムは進化すべきだ」 | 改革派として |
| 「私が手配しよう」 | 助けを申し出る時 |

**話し方の特徴**：
- 知的。穏やか
- 質問が多い（好奇心）
- 敬語と砕けた口調を使い分ける

### アイリーン

| 口癖 | 使用場面 |
|------|----------|
| 「私たちは300年間、答えを出せなかった」 | 終焉派として |
| 「それが答えではないのか」 | 逆説的な問い |
| 「あなたを守ることはできない。でも——」 | 静かな支援 |

**話し方の特徴**：
- 静か。声が小さい
- 言葉が少ない。でも、重い
- 微笑むことがある（レイに対して）

## 会話サンプル（キャラブレ確認用）

### レイとユキ

```
ユキ：「あんた、まだそんなことやってるの」
レイ：「ああ」
ユキ：「フェアリーに逆らっても、何も変わらないのに」
レイ：「……」
ユキ：「……なんで、何も言わないの」
レイ：「お前が言いたいのは、それじゃないだろ」
ユキ：「——」
```

### レイとカイ

```
カイ：「レイ、俺は限界かもしれない」
レイ：「そうか」
カイ：「……それだけか？」
レイ：「お前が決めることだ。続けるも、やめるも」
カイ：「——お前は、なんで平気なんだ」
レイ：「……俺にも、わからない」
```

### レイとリン

```
リン：「レイさん、私……何かできることありますか」
レイ：「いつも通りでいい」
リン：「いつも通り……」
レイ：「お前がいると、みんな落ち着く。それでいい」
リン：「……私、何もしてないのに」
レイ：「何かしてる。お前が気づいてないだけだ」
```

### レイと老師

```
老師：「小僧、お前は何を見た」
レイ：「……わからない。でも、何かがおかしい」
老師：「考えろ。自分で」
レイ：「教えてくれないのか」
老師：「教えても意味がない。お前が選ばなきゃ、何も始まらん」
```

### レイとセラフィム

```
セラフィム：「お前の『俺』に、何の価値がある」
レイ：「……」
セラフィム：「測れない。売れない。誰の役にも立たない」
レイ：「だから何だ」
セラフィム：「——」
レイ：「価値があるかどうかは、俺が決める」
```

## 注意事項

1. **レイは説教しない**——他人の選択を尊重する
2. **ユキは感情を抑える**——でもレイの前では漏れる
3. **カイは弱音を吐く**——でも諦めない
4. **リンは謙遜する**——でも逃げない
5. **老師は答えを教えない**——考えさせる
6. **セラフィムは冷たい**——でも恐怖を隠している
7. **ソウマは完璧**——でも空虚
