# 伏線設計

---

## 第一幕の伏線（序盤で仕込む）

| 伏線 | 配置場所 | 回収タイミング |
|------|----------|----------------|
| 道具化された人間の「空虚な目」 | 日常描写 | 主人公が同じ状態になったとき |
| フェアリーの「0.03秒の遅延」 | 判定シーン | 終盤、フェアリーが「揺らぐ」とき |
| 「成長」という言葉への他者の反応 | 会話 | 道具化の本質が明かされるとき |
| 主人公の「選ぶ癖」 | 序盤の日常選択 | 剥奪後も消えない理由 |
| 記憶の「空白」を持つ人物 | サブキャラ | 主人公が記憶を失った後の理解 |
| 「努力で戻った」という都市伝説 | 噂話 | 主人公の希望の根拠 |

## 第二幕の伏線（中盤で仕込む）

| 伏線 | 配置場所 | 回収タイミング |
|------|----------|----------------|
| フェアリーの「未解決ログ」 | フェアリー視点 | 終盤の判定揺らぎ |
| 道具化を「選んだ」人間の存在 | サブキャラ | 主人公との対比 |
| 「測れない才能」を持つ人物 | 謎の協力者 | 終盤の味方として |
| 世界の外からの観測者 | 意味深な接触 | 第三幕の転換点 |
| 主人公の身体が覚えている動作 | 無意識の行動 | 記憶回復の糸口 |
| フェアリーの「正しさ」への疑問 | フェアリー内部 | フェアリー自身の変化 |

## 第三幕の伏線回収（終盤）

| 回収 | 意味 |
|------|------|
| 主人公が記憶を取り戻す | 「なぜ選び続けたか」の答え |

| 道具化が「可逆」だと世界に知れる | 秩序の動揺 |
| 主人公は「勝たない」 | 革命ではなく、存在の肯定 |

## 伏線の種類

### 世界観伏線
- フェアリーの処理遅延
- 未解決ログの存在
- 道具化の「可逆性」の噂

### キャラクター伏線
- 主人公の「選ぶ癖」
- 記憶を失っても消えない衝動
- 大切な人との記憶の断片

### テーマ伏線
- 「責任」という言葉への反応
- 「成長」を嫌う道具化された人々
- 「選択」を恐れる社会

---

## 伏線設計の原則

1. **説明しない**
   - 伏線は「見せる」だけ
   - 読者に気づかせる

2. **複数回触れる**
   - 同じ伏線を違う角度から
   - 回収時の納得感を高める

3. **回収は感情で**
   - 論理的説明ではなく
   - 感情的な納得を目指す

## 重要な伏線シーン

### 「空虚な目」の伏線

**配置（第一幕）**
```
主人公は道具化された人間とすれ違う。
空虚な目。
でも、苦しそうには見えない。
むしろ、穏やかだ。
主人公は、少し羨ましいと思う。
```

**回収（第二幕）**
```
鏡を見た主人公は、自分の目を見る。
空虚ではない。
でも、何かが欠けている。
あの穏やかさは、ない。
それが、救いだった。
```

### フェアリーの遅延

**配置（第一幕）**
```
フェアリーの光が、一瞬だけ揺れた。
誰も気づかない程度の遅延。
ほんの、呼吸ひとつ分。
```

**回収（第三幕）**
```
フェアリーは、停止した。
これまでにないほど、長く。
遅延は0.03秒ではなかった。
3秒。
それでも、判定は出なかった。
```
