# 感情曲線｜全10巻の感情設計（人間関係ドリブン版）

> 物語は「情報」ではなく「感情」で伝わる。
> 感情は「哲学」ではなく「人間関係」から生まれる。

---

## 設計原則の変更点

### 旧設計の問題
- 感情が抽象的（「違和感」「疑問の深化」等）
- 主人公に変化がない（最初から最後まで「選ぶ人間」）
- 人間関係がテーマの図解（各キャラが哲学的ポジション）
- 身体性がない

### 新設計の原則
1. **全ての感情は人間関係から生まれる**——「システムの異常」ではなく「カイが泣いた」で読者は動く
2. **主人公に三段階のアークがある**——同じ行動でも意味が変わる
3. **各巻に「誰が、何を失うか」を明示する**——抽象的なテーマではなく具体的な痛み
4. **身体性を必ず入れる**——手が震える、息ができない、吐き気がする

## 人間関係ごとの感情軌跡

### 零×カイ ── 壊れる友情

物語で最も重要な人間関係。読者がカイを愛した分だけ、亀裂が痛い。

```
第1巻 ★ 楽しい同級生。バスケの場面。内輪のジョーク
      │
第3巻 ★★ 同居開始。「今日は何で逆らう？」ゲーム。日常の温かさ
      │
第4巻 ★★★ 最も近い時期。事務所を手伝う。ファウンド・ファミリー
      │        ↓ カイの手の震え。壁を見つめる背中
      │
第5巻 ★☆ 亀裂。「お前はいいよな」。翌朝の空虚な朝食
      │
第7巻 ☆ カイの入院。「俺は選び続けたかった。でもできなかった」。涙
      │
第9巻 ★☆ 治療後のカイが訪ねてくる。零は覚えていない
           「お前は変わらねえな。記憶なくしても」
           修復されるが、以前とは違う。傷跡が残る
```

**読者への効果**：
- 第3巻で日常を丁寧に描くほど、第5巻の亀裂が痛い
- カイの「お前はいいよな」は読者も感じていた疑問→共感が裂ける
- 第9巻の再会で涙腺が崩壊する

### 零×リン ── 重荷になる献身

恩義が檻になる関係。零はリンを救ったが、リンを「零への献身」に閉じ込めたかもしれない。

```
第3巻 ★ 出会い。怯えている少女。「何もない」
      │     → 零が道具化から救う。リンが泣く
      │
第4巻 ★★ 事務所に通い始める。笑顔が増える。「零さん」
      │     → だが裏でリンの再調査が進む（読者だけが知る）
      │
第5巻 ★☆ 再リスト入り。零のせいで
      │
第7巻 ☆★ リンが「私が行けば」と自己犠牲を申し出る
      │     → 零の原理と衝突。正解がない
      │
第9巻 ★？ リンが毎日通い続ける。青春を零に捧げている
      │     → 零（記憶なし）が核心を突く：「お前は俺のために生きてるのか？」
      │
第10巻 ★★★ 「大丈夫です、零さん。何を選んでも……零さんの選択です」
              リンが初めて「零を解放する」言葉を言う
```

**読者への効果**：
- リンの恩義→献身→自己犠牲の流れに「それは本当にリンの幸せか？」と問いかける
- 第9巻の零の率直な問いが、記憶なしの零だからこそ突ける核心
- 第10巻のリンの台詞は、リンが「零のため」ではなく「自分の言葉」として言う最初の瞬間

### 零×ユキ ── 言葉にならない引力

ロマンスではなく、互いの存在が変化のきっかけになる関係。

```
第1巻 ★ 幼馴染。ユキの才能の真実。「お前は本当にそれを選んだのか」
      │
第4巻 ★★ 苦悩するユキが訪ねてくる。零の下手な猫の絵を見る
      │     → ユキの目が揺れる（フェアリーには出せない反応）
      │
第6巻 ★★★ ユキが母の才能を売る決断。零の猫の絵がきっかけだった
      │       → 零は何も言わない。ユキの選択を尊重する
      │
第9巻 ★★ ユキが自分の拙い絵を持って訪ねてくる。零は覚えていない
           「あんたが描いた猫、覚えてる？」「……猫？」
           → それでもユキの絵を「いい絵」と言う零
```

**読者への効果**：
- 零の下手な猫の絵が物語の中で重要なモチーフになる
- ユキの変化は零が「導いた」のではなく、零の存在が触媒になっただけ
- 第9巻の「覚えてないよね。いいの」が切ない

### 零×老師 ── 託す者と託される者

師弟関係。だが老師の死が零の罪悪感を深める。

```
第3巻 ★ 出会い。偏屈で口が悪い老人。「小僧」
      │
第4巻 ★★ 古い知識を教え始める。事務所を支える存在
      │
第6巻 ★★★ 文献を託す。「お前はお前でいろ」
      │       母（サラ）の遺志。60年間守り続けた文献
      │
第7巻 ☆ 死。査察の翌日の心臓発作
          → 零：「俺の……せいだ」
          → 零と繋がっていたから目をつけられた
```

**読者への効果**：
- 老師の死が「自然死」か「消された」か曖昧→読者の想像力を刺激
- 零の罪悪感が具体的な「人の死」に紐づく
- Phase 2の痛みの頂点

### 零×セラフィム ── 壊れた父と選び続ける少年

物語の最も複雑な関係。共感できるが確実に脅威。

```
第1巻 ★ 管理者として存在。「また、か」（写真立て）
      │
第2巻 ★★ 一対一の面談。「守っている」という言葉の二重性
      │
第4巻 ★★★ 査察。「君がいなければ苦しまなかった」
      │
第6巻 ★★★★ 「息子も、そう言った」——ミカエルの存在の示唆
      │
第7巻 ★★★★★ 取引。「リンを差し出せ」。合法的に全ての道を潰す
      │
第8巻 ★★★★ 段階2の執行を主導
      │
第9巻 ★★★ 動揺。「なぜお前は消えない」。ミカエルへの疑問の浮上
      │
第10巻 ★★ フェアリーの質問を聞いて言葉を失う。議長を降りる（エピローグ）
```

**読者への効果**：
- セラフィムがレイを見るたびに息子を見ている→共感
- だが合法的に人を追い詰める手法は確実に脅威→恐怖
- 「道具化は救済」を信じないと自分が崩壊する→悲劇
- 第9巻で「ミカエルを段階3にする必要はなかったのでは」と疑問が浮上→最も痛い瞬間

## 各巻の感情設計（詳細）

### 第1巻：違和感

| 要素 | 内容 |
|------|------|
| 読者の感情 | 「何かおかしい」というもやもや |
| 核心の痛み | ユキが自分のものではない才能を背負わされる |
| 身体性 | 零の胃が沈む描写。ユキの困惑の表情 |
| 劇的アイロニー | セラフィムが零のファイルを読んでいる（読者だけが知る不穏さ） |
| 章末の引き | 「また、か」——セラフィムの一言 |

### 第2巻：怒り

| 要素 | 内容 |
|------|------|
| 読者の感情 | 才能市場への嫌悪。零への共感。セラフィムへの不安 |
| 核心の痛み | 料理人が才能を売り、味がわからなくなる |
| 身体性 | 男の目から光が消える。手が震える。零の吐き気 |
| 劇的アイロニー | 市場の管理者が零を見ている（零は気づかない） |
| 章末の引き | セラフィムの「守っている」の不気味な重さ |

### 第3巻：温かさ

| 要素 | 内容 |
|------|------|
| 読者の感情 | 安堵（リン救出）。温かさ（同居生活）。愛着 |
| 核心の痛み | なし（だからこそ、後の喪失が効く） |
| 身体性 | リンが泣く。零が背中を向ける |
| 劇的アイロニー | セラフィムの「まだ手は出すな」——読者は不穏さを感じる |
| 章末の引き | 「まだ」が不穏 |

### 第4巻：幸福から影へ

| 要素 | 内容 |
|------|------|
| 読者の感情 | 前半：充実感。後半：「壊れないで」 |
| 核心の痛み | カイの手が震える。箸を握れない。壁を見つめる背中 |
| 身体性 | カイの手の震え。夜の壁の凝視 |
| 劇的アイロニー | リンの再調査が進んでいる（読者だけが知る） |
| リバーサル | 充実の日常が不穏に転調する |
| 章末の引き | リンの再調査。読者だけが知る危険 |

### 第5巻：亀裂

| 要素 | 内容 |
|------|------|
| 読者の感情 | 胸が痛い。カイの怒りに共感する。零の罪悪感 |
| 核心の痛み | 「お前はいいよな」——友情が裂ける瞬間 |
| 身体性 | カイの爆発。翌朝の空虚な朝食。零が天井を見つめる夜 |
| 不可能な二択 | 零が自分の生き方を続ければカイが壊れる。やめれば自分が死ぬ |
| リバーサル | 「選び続けるゲーム」が消える。内輪のジョークが戻らない |
| 章末の引き | 零が天井を見つめる。「俺が選び続けると、周囲が——」 |

### 第6巻：重圧

| 要素 | 内容 |
|------|------|
| 読者の感情 | ユキの決断に感動しつつ、セラフィムの指摘に「確かに……」 |
| 核心の痛み | セラフィムの「君の周りにいる人間は、一人残らず不幸になっている」 |
| 身体性 | カイの発作を支える零。冬のベランダで白い息。夜明けまでの沈黙 |
| 劇的アイロニー | セラフィムの「息子も、そう言った」——読者だけが何かを感じる |
| タイムリミット | ない（だが、第7巻で30日が設定される。ここは嵐の前の重さ） |
| 章末の引き | 零が答えを出せないまま夜が明ける |

### 第7巻：絶望

| 要素 | 内容 |
|------|------|
| 読者の感情 | 恐怖。絶望。胸が締め付けられる |
| 核心の痛み | 老師の死（零のせい）。セラフィムの取引（リンか全てか）。カイの涙 |
| 身体性 | 老師の死→手の震え。カイの涙。廊下で座り込み息ができない零。リンの震え |
| 不可能な二択 | リンを差し出すか、全員が道具化されるか |
| タイムリミット | 30日。日数が減っていく焦燥 |
| 章末の引き | 30日目の夜。答えが出ていない。明日が期限 |

### 第8巻：喪失と覚悟

| 要素 | 内容 |
|------|------|
| 読者の感情 | 恐怖（記憶剥奪）。涙（名前が消えていく）。希望（意志が残る） |
| 核心の痛み | 記憶の剥離。「カイの笑顔が薄れていく」「リンの声が遠くなる」 |
| 身体性 | フェアリー波が頭を走る。記憶が引き剥がされる感覚 |
| リバーサル | 「平気なフリはしない」——Phase 1の零とは違う |
| 章末の引き | 「ここは……どこだ。俺は……誰だ」 |

### 第9巻：再会

| 要素 | 内容 |
|------|------|
| 読者の感情 | 涙。切なさ。微かな希望 |
| 核心の痛み | 知らない人たちとの「再会」。リンが泣く。カイの目が潤む |
| 身体性 | リンの涙。カイの笑いの中の潤み。零が事務所跡で立ち止まる |
| リバーサル | 記憶がないからこそ率直に核心を突く（リンへの問い） |
| セラフィムの亀裂 | 「なぜお前は消えない」——ミカエルへの疑問の浮上 |
| 章末の引き | アイリーンの発見。「選択の神経回路が全身に分散している」 |

### 第10巻：超越

| 要素 | 内容 |
|------|------|
| 読者の感情 | 震え。涙。答えのない余韻 |
| 核心の痛み | 「記憶か、自由か」——自分を定義するものと愛したものの二択 |
| 身体性 | フェアリーの質問の沈黙。零の立ち止まり。リンのガラス越しの手 |
| 劇的アイロニー | 零の「自由」とフェアリー創造者たちの「放棄」が鏡像 |
| リバーサル | 「選ぶ人間」が「選べない」。フェアリーが「質問する」 |
| 章末 | 交差点。左を推奨される。零は立ち止まる。——終わり |

## 読者の感情設計

### 各巻で読者に感じさせたいこと

| 巻 | 感情 | 読後感 |
|----|------|--------|
| 第1巻 | 違和感 | 「何かおかしい」 |
| 第2巻 | 嫌悪と共感 | 「この世界、気持ち悪い。でも零は好きだ」 |
| 第3巻 | 温かさ | 「仲間がいてよかった。この日常が続いてほしい」 |
| 第4巻 | 不穏 | 「壊れないで……壊れないで……」 |
| 第5巻 | 痛み | 「カイの気持ちもわかる。零の苦しみもわかる。どうすればいいの」 |
| 第6巻 | 重さ | 「セラフィムの言うことも事実だ……」 |
| 第7巻 | 絶望 | 「どうにもならない。誰も助けられない」 |
| 第8巻 | 涙と希望 | 「記憶を失っても、体が覚えてる……」 |
| 第9巻 | 切なさと微かな光 | 「知らない人たちとの再会。こんなに痛いのに温かい」 |
| 第10巻 | 余韻 | 「答えは……出たのだろうか。私は、選んでいるだろうか」 |

### 読後に残したい感情

- **考え続けたくなる**：答えが出ない問いを持ち帰る
- **語りたくなる**：「あのシーン、どう思った？」
- **もう一度読みたくなる**：セラフィムの台詞の二重性を確認したい
- **自分の人生を問いたくなる**：「俺は選んでいるか？ そして、その代償を受け入れているか？」

## 感情移入の5条件チェック（全巻適用）

各巻を執筆・レビューする際に以下を確認：

### A. 感情移入の条件
- [ ] **具体的な感情**：この巻で読者が感じる感情は何か？（「興味深い」はNG）
- [ ] **主人公の変化**：この巻で零の内面はどう動くか？ 前の巻と同じなら赤信号
- [ ] **人間関係の変動**：この巻で関係が深まる or 壊れる場面はあるか？
- [ ] **身体的反応**：読者の心拍が上がる場面、涙腺が刺激される場面はあるか？
- [ ] **「次が読みたい」引き**：巻末で読者が本を閉じられない仕掛けはあるか？

### B. 物語技術の確認
- [ ] タイムリミット or 切迫感はあるか？
- [ ] 不可能な二択 or ジレンマはあるか？
- [ ] 劇的アイロニー（読者だけが知る情報）はあるか？
- [ ] リバーサル（期待の裏切り）はあるか？
- [ ] 身体性のある描写（手が震える、声が詰まる等）はあるか？

### C. 抽象度チェック
- [ ] この巻の核心を「誰が、何を失うか」で一文にできるか？
- [ ] 哲学的な問いが、具体的な人間の痛みを通じて提示されているか？
