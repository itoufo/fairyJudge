## 第1章「審判の日」感情曲線

### レイの感情

```
      違和感
        │
    ★───────★ ユキの判定を見る
   ／         ＼
  ／           ＼     ★ 「お前は本当に、それを選んだのか？」
 ／             ＼   ／
★               ★─★
日常            判定後    帰り道

感情：平穏 → 違和感 → 言語化できない疑問
```

### ユキの感情

```
        ★ 興奮（才能を認められた！）
       ／│
      ／ │
     ／  ★ 微かな不安（なぜ私が？）
    ／   │
★──★    │
緊張      レイの言葉を聞いて
          ↓
          ★ 心の奥に残る棘

感情：緊張 → 喜び → 棘が刺さる
```

### キーモーメント

1. **才能判定の瞬間**
   - ユキ：「私には才能がある！」（表面）
   - ユキ：（でも、これは本当に私の？）（心の奥）

2. **レイの違和感**
   - 言葉にならない
   - 「何かがおかしい」としか言えない
   - **読者もこの時点では言語化できない**

3. **帰り道の問い**
   - 「お前は本当に、それを選んだのか？」
   - ユキには意味がわからない
   - **だが、この言葉は10年後に意味を持つ**

## 感情設計の原則

### 1. 対比の法則

| 対比 | 効果 |
|------|------|
| 静と動 | 第8章の静けさが第9章の激しさを際立たせる |
| 希望と絶望 | 第4章の希望が第7章の喪失を深くする |
| 個と全体 | レイの孤独が、仲間との絆を意味深くする |

### 2. 伏線と回収

| 伏線 | 回収 |
|------|------|
| 第1章：「本当に選んだのか」 | 最終章：ユキが「選ぶ」 |
| 第3章：「いつか売るのかな」 | 第8章：ユキが才能を売る |
| 第5章：0.03秒の遅延 | 最終章：フェアリーが「質問」 |
| 第6章：「逃げられる」 | 最終章：「ここで選び続ける」 |

### 3. 感情の残響

各章の終わりに「問い」を残す：

| 章 | 残る問い |
|----|----------|
| 第1章 | 「才能」って本当に自分のもの？ |
| 第2章 | 「選ぶ」ことは間違いなのか？ |
| 第3章 | 「魂」はどこに行った？ |
| 第4章 | なぜレイだけが「平気」なのか？ |
| 第5章 | フェアリーは本当に「正しい」のか？ |
| 第6章 | 「逃げる」ことは「選ぶ」ことか？ |
| 第7章 | 「託す」とは何か？ |
| 第8章 | 「選んだ」感覚とは何か？ |
| 第9章 | 「俺」に価値はあるのか？ |
| 最終章 | 答えは——出たのか？ |

## 読者の感情設計

### 各章で読者に感じさせたいこと

| 章 | 感情 | 読後感 |
|----|------|--------|
| 第1章 | 違和感 | 「何かおかしい」 |
| 第2章 | 共感 | 「わかる、その気持ち」 |
| 第3章 | 嫌悪 | 「この世界、気持ち悪い」 |
| 第4章 | 安堵 | 「仲間がいてよかった」 |
| 第5章 | 好奇心 | 「なぜレイだけ？」 |
| 第6章 | 緊張 | 「危ない、逃げて」 |
| 第7章 | 悲しみ | 「老師……」 |
| 第8章 | 静謐 | 「嵐の前の静けさ」 |
| 第9章 | 震え | 「それでも、止まらないのか」 |
| 最終章 | 余韻 | 「答えは……出たのだろうか」 |

### 読後に残したい感情

- **考えたくなる**：答えが出ない問いを持ち帰る
- **語りたくなる**：誰かと話し合いたくなる
- **もう一度読みたくなる**：伏線を確認したくなる
- **自分の人生を問いたくなる**：「俺は選んでいるか？」
